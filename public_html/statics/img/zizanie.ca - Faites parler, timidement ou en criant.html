Quel type de page choisir?

Si vous souhaitez que les visiteurs réagissent à votre sujet à l'aide de commentaires, sélectionnez le type de page «Commentaires».

Si vous choisissez le type «Commentaires (Pour/Contre)», il y aura 2 colonnes de commentaires selon que vous soyez pour ou contre le sujet.

Le type «Bouton Tweet», quant à lui, laisse le visiteur compléter votre sujet et l'envoyer sur Twitter.
